Purpose & Vision

Our Mission

The Alma 31 Study app was created to provide a comprehensive, accessible, and engaging platform for studying one of the most instructive chapters in the Book of Mormon. Our goal is to help individuals, families, and teachers gain deeper insights into the eternal principles taught through Alma's encounter with the Zoramites.

Technology & Design

Technical Stack

Frontend Technologies

  • HTML5: Semantic markup for accessibility and SEO
  • CSS3: Modern styling with custom properties and responsive design
  • JavaScript (ES6+): Interactive features and dynamic content

Design Principles

  • Mobile-First: Responsive design starting with mobile optimization
  • Progressive Enhancement: Core functionality works without JavaScript
  • Accessibility (WCAG 2.1): Screen reader compatible with keyboard navigation
  • Performance: Optimized loading with lazy loading and minimal dependencies

Progressive Web App Features

  • Service Worker: Offline caching for core content
  • Web App Manifest: Installable on devices
  • Responsive Icons: Properly sized for all platforms
  • Theme Integration: Matches device preferences

Design Inspiration

The visual design is inspired by churchofjesuschrist.org, incorporating the clean, reverent aesthetic that characterizes official Church resources. This includes:

  • Church blue (#002E5D) as the primary color
  • Clean typography with serif fonts for body text
  • Generous white space for improved readability
  • Professional layout that emphasizes content over decoration

Browser Support

  • Edge 90+ (desktop and mobile)
